# 코드 리뷰

#### 💻 프로젝트명
  대관령 음악제

#### 🛠 사용한 기술
  HTML, SCSSS, JQUERY

#### 🎯 모작 이유
  마우스 커서 제어, 헤더 스타일 변경 등 다양한 jQuery 기능을 직접 구현해보고 싶었고, SCSS를 활용해 구조적인 코드 작성을 연습해보고 싶어서 이 사이트를 모작하게 되었습니다.

#### 😱 어려웠던 점
 - 하나의 이벤트 리스터 안에 여러 기능을 동시에 제어하는 것이 어려웠습니다.
 - grid-template 속성을 사용하여 여러 아이템을 일정한 비율로 배치하는 것이 생각보다 까다로웠습니다.
